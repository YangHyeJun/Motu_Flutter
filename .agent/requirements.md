# 동작 사양

## 스플래시 화면
- 스플래시 화면에서 연결중이라는 의미로 하단에 원형 점 세 개가 1, 2, 3번 깜빡이도록 구현한다.

## 매수
- 매수 버튼을 클릭하면 매수 화면이 새로 추가되고, 해당 화면에서는 매수/매도의 체결량을 보여준다.
- 각 가격별 체결량은 실시간성을 보장해야한다.
- 매수할 주식의 개수를 입력받을 수 있게 한다. 현재 내가 매수할 수 있는 금액 한에서 입력받을 수 있게 한다.
- 매수의 사용자 경험은 빠르게 이루어져야 하도록 한다.
- 매수 할 금액은 실시간 금액과 동일하게 한다.
- 체결량을 보여줄 때 가격도 보여준다. 각 가격별 체결량에서, 해당 가격을 누르면 매수 금액이 해당 금액으로 설정된다.
- 매수 버튼을 최종적으로 클릭하면 실제로 매수되도록 한다.

## 매도
- 매수와 같은 사양을 유지한다.

## 소켓 연결
- 실시간성 변화가 있는 모든 UI에는 소켓을 연결한다. (코스피, 나스닥 지수 표시, 주식 목록에서 가격 표시, 주식 자세히 보기 화면 등)
- 가격이나 체결량을 표시하는 화면은 기본값으로 실시간 소켓 연결이 유지되어야 하며, 별도 요청이 없어도 주식 목록/검색 결과/즐겨찾기/상세 화면 모두 실시간 가격 변동이 반영되어야 한다.
- 기능 수정 전후에는 가격 표시 화면의 소켓 구독 상태와 실시간 반영 여부를 항상 확인한다.
- 연결이 끊긴 경우, 연결이 끊겼음을 사용자에게 알리고 연결 재시도를 할 수 있게 한다.

## 연결 재시도
- 재시도 버튼 클릭시 해당 위젯에 대해 재시도를 할 수 있게 한다. 전체 모든 UI를 재시도 하지 않는다.
- 화면 전체를 아래로 스크롤하여 재시도를 하는 경우에는 전체 재시도를 한다.
- 재시도 버튼과 UI의 배치를 어색하지 않는곳에 표시한다.
- 각 위젯별 재시도 시간을 표시한다.


## 한 개의 주식 자세히 보기 (detail_screens.dart)
- 주식 가격 변동에 대해 실시간성을 보장한다.
- 차트 변동에 대해 실시간성을 보장한다.
- 차트에는 월/일/시간이 표시된다. 년도가 바뀌는 경우에는 년도도 표시한다.

## 주식 검색
- 검색 바에 영/한국어를 입력할 수 있다.
- 미국 주식도 한국어로 검색이 가능하다.
- 검색어가 있는 경우 우측에 X 표시가 뜨고, X 버튼을 누르면 검색어가 사라진다.
- 검색어에 대한 검색 결과는 실시간으로 보여준다.
